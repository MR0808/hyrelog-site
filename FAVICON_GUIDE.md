# Favicon Guide for HyreLog

To create a comprehensive favicon that works across all devices and platforms, you'll need to provide the following:

## Required Assets

### 1. **Master Icon (SVG or High-Resolution PNG)**

-   **Format**: SVG (preferred) or PNG
-   **Size**: At least 512x512 pixels (1024x1024 recommended)
-   **Design**: Simple, recognizable version of your logo
-   **Background**: Transparent or solid color
-   **Colors**: Should work on both light and dark backgrounds

### 2. **Square Version**

-   Your logo should work well in a square format
-   If your logo is horizontal, create a square variant or use an icon version

## What I Need From You

1. **The master icon file** (SVG or high-res PNG)

    - Preferably SVG for scalability
    - Or a PNG at 1024x1024 pixels minimum

2. **Color preferences**:

    - Primary brand color
    - Whether you want a monochrome version for dark mode

3. **Any specific design requirements**:
    - Should it include text or just the logo mark?
    - Any specific styling or effects?

## What I'll Generate

Once you provide the master icon, I'll create and configure:

-   `favicon.ico` (16x16, 32x32, 48x48 sizes)
-   `icon.png` (various sizes: 192x192, 512x512)
-   `apple-touch-icon.png` (180x180 for iOS)
-   `manifest.json` for PWA support
-   Proper metadata in `layout.tsx`

## Quick Start

If you have your logo ready:

1. Place the master icon file in `/public/` directory
2. Let me know the filename
3. I'll generate all the required favicon files and configure them

## Alternative: Online Tools

If you prefer to generate favicons yourself, you can use:

-   **RealFaviconGenerator**: https://realfavicongenerator.net/
-   **Favicon.io**: https://favicon.io/

These tools will generate all the necessary files from a single source image.
